# Personal Portfolio Website

This repository contains the source code for my professional portfolio website, hosted at [denizdalbasi.github.io/website/](https://denizdalbasi.github.io/website/). The site shows everything about my work, my projects, my technical skills, and the path I’ve taken in my professional journey.

---

### Objectives
* **Project Showcase:** A space to highlight the projects I’ve built and the contributions I’m most proud of
* **Technical Resume:** Providing a clear overview of my skills and professional experience
* **Clean UI:** A clean, responsive design that feels intuitive on any device

### Technical
* **HTML & CSS:** For the main design of the website
* **JavaScript:** Implementing interactive simulations and the page effects
* **GitHub Pages:** Used to make the website public
