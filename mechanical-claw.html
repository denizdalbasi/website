Mechanical Claw

Embedded Systems & Robotics Project

Arduino C++ SolidWorks Sensors

Project Description

For this project, I designed and programmed a robotic mechanical claw capable of autonomous object detection and handling. The goal was to create a system that could execute precise gripping sequences while managing power constraints and real-time sensor data.

Technical Implementation:

  • Embedded Control: Wrote the firmware in C++ for an Arduino microcontroller to handle motor PWM signals.
  • Sensor Logic: Implemented a feedback loop using ultrasonic/IR sensors to detect when an object is within the optimal gripping range.
  • Mechanical Design: Used CAD software to model the claw assembly, ensuring the gear ratio provided enough torque to lift small objects.
